Button

Buttons are used in every application as a submission tool for users.

Demo

Solid buttons look heavy and are used for primary actions.
They direct the user's attention to the suggested action.


Outline buttons are used for secondary actions.
They reduce visual noise on the page.


Flat buttons are used for tertiary actions.
They can be used inline, in navigation features, and more.

Grouped Buttons

Grouped buttons are appropriate for filters, or when there are several related decisions that the user can apply.
Active filters are solid buttons with emboldened text, while inactive filters are effectively flat buttons.
