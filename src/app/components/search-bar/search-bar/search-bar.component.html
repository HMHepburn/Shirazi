Typography

The placeholder and filler text should have the same font - however the placeholder text should be a much fainter colour to indicate it is temporary.

DO: Placeholder is light and indicates that the input can be entered.

DON'T: Placeholder is the same colour as text - it is unclear to the user whether the field is already filled.

Size

The searchbar's size is variable depending on the length of text that may be entered, the size of surrounding components, and more.

Generally, the size should adapt to different screen sizes and look appropriate.
